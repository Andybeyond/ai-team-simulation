## Project Structure
Each project follows a standardized structure:
```
project_name/
├── docs/              # Project documentation
├── src/              # Source code
├── tests/            # Test files
└── README.md         # Project-specific documentation
```

## Adding New Projects
When adding a new project:
1. Create a new directory under /projects
2. Follow the standard project structure
3. Update this README.md with project details
